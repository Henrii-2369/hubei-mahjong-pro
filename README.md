# 🀄️ 湖北麻将计算器 (纯数学版)

一个专为湖北麻将（红中癞子杠）设计的移动端辅助助手。使用本地算法实时计算牌效和最佳出牌策略。

## ✨ 功能特点

*   **移动端优先**: 专为手机设计的 UI，大按钮，便捷操作。
*   **湖北规则**: 严格遵循红中癞子杠玩法（2/5/8做将）。
*   **纯本地计算**: 无需联网，无需 API Key，保护隐私，响应迅速。
*   **iOS/PWA 支持**: 可以添加到主屏幕，像原生 App 一样使用。

## 📱 iOS 部署指南 (添加到主屏幕)

本项目是一个纯前端 Web 应用 (Single Page Application)。

### 1. 部署 (选择一种方式)

#### 方式 A: Vercel (推荐)
1. Fork 本仓库到你的 GitHub。
2. 登录 [Vercel](https://vercel.com)。
3. 导入项目并直接点击 Deploy (无需配置环境变量)。

### 2. 在 iPhone 上安装
1. 打开 Safari 浏览器。
2. 访问你部署好的网址。
3. 点击底部中间的 **分享按钮** (Share Icon)。
4. 点击 **"添加到主屏幕" (Add to Home Screen)**。
5. 现在你可以离线使用计算器了。

## 🛠️ 本地开发

1. 克隆项目
2. 安装依赖: `npm install`
3. 启动: `npm start`
